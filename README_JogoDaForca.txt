# JogoDaForca
Feito por: Lígia Morais de Souza
O objetivo do programa se baseia em um jogo da forca habitual. O tema em comum entre todas as palavras é informática, e cabe ao usuário adivinhar qual é a palavra da vez, antes que suas tentativas se esgotem.

## O que foi utilizado
**Linguagem de Programação:** C#
**Framework:** .NET
**Versionamento:** GIT
**Gestão de Repositórios:** GitHub

## Etapas Implementadas
**Entrada do Jogador:** Captura as tentativas do jogador 
**Validação das Jogadas:** Verifica se a posição escolhida é válida e se já está ocupada.
**Atualização do Tabuleiro:** Atualiza o estado do tabuleiro após cada jogada.
**Exibição do Resultado: Mostra o estado atual do jogo após cada movimento.
**Verificação de Vitória:** Avalia se algum jogador ganhou após cada jogada.

## Backlog
**Adicionar Dificuldade:** Implementar um modo de jogo contra com diferentes níveis de dificuldade.
**Salvar e Carregar Jogos:** Permitir que os jogadores salvem seu progresso e carreguem jogos anteriores.
**Temas e regras:** Exibir regras do jogo no código e criar opções de escolha de temas variados.

## Conclusão
O jogo da velha é uma ótima maneira de passar o tempo entre amigos, colegas de turma e até mesmo consigo mesmo.  A estrutura do código é simples, porém uma fonte de aprendizado e inspiração para ajustes e modificações no futuro.
